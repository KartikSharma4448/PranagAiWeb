# PranagAiWeb

PranagAiWeb is a Vite-based React frontend built with TypeScript, Tailwind CSS, Radix UI, and modern testing tooling. It is designed as a polished web UI layer for the Pranag AI ecosystem.

## Highlights

- Fast Vite development workflow
- React + TypeScript component architecture
- Tailwind CSS styling with Radix UI primitives
- Router-driven UI with reusable feature modules
- Vitest and Playwright tooling available for validation

## Tech Stack

- Vite
- React 18
- TypeScript
- Tailwind CSS
- React Router
- Radix UI
- TanStack React Query
- Vitest
- Playwright

## Getting Started

1. Install dependencies:

	```powershell
	pnpm install
	```

2. Start the development server:

	```powershell
	pnpm dev
	```

3. Build for production:

	```powershell
	pnpm build
	```

4. Run checks:

	```powershell
	pnpm lint
	pnpm test
	```

## Repository Notes

- `src/` contains the application code.
- `public/` stores static assets.
- `vite.config.ts` controls the build pipeline.
